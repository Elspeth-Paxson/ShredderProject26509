Welcome

Welcome to our senior design project website!

U of A EDC & Catalyst Studios create 50 kg of waste every semester This waste can be recycled; however, the sheer volume of waste is difficult to store and transport. To be recycled, the PLA waste needs to be shredded. Our teams mission is to iterate on a previous project in creating a shredder specifically geared towards makerspace use.

Project Goals

  • Improve shredding performance
  • Increase system reliability
  • Enhance user safety
  • Support plastic recycling efforts

About the Project

The 3D Plastic Shredder Version II is a senior engineering capstone project focused on converting 3D plastic prints into a compactable material. This project improves upon a previous team's designs to create a more efficient and reliable shredding system while still being user-friendly and safe!
