About the Project

The 3D Plastic Shredder Version II is a senior engineering capstone project focused on converting plastic waste into reusable shredded material. This project improves upon previous designs to create a more efficient, reliable, and user-friendly shredding system.

Project Goals

  • Improve shredding performance
  • Increase system reliability
  • Enhance user safety
  • Support plastic recycling efforts
